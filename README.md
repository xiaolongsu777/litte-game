# 小游戏合集

一个纯静态 HTML/CSS/JS 小游戏合集，目前包含：

- 扫雷
- 金矿大亨

## 本地打开

直接打开 `index.html` 即可进入游戏入口页。

## Vercel 部署

这个项目不需要构建步骤，可以直接作为静态网站部署到 Vercel。

1. 将项目提交到 GitHub 仓库。
2. 在 Vercel 新建项目并导入该仓库。
3. Framework Preset 选择 `Other`。
4. Build Command 留空。
5. Output Directory 留空，使用项目根目录。
6. 部署完成后，把 Vercel 的 HTTPS 链接发给朋友。

## iPhone 分享测试清单

- 用 iPhone Safari 打开 Vercel 链接。
- 入口页能进入“扫雷”和“金矿大亨”。
- 金矿大亨的金币按钮、商店、宝石工坊和运输动画不重叠。
- 点击金币有反馈，购买和出售按钮可正常点击。
- 刷新页面后，本机浏览器存档仍保留。
- 可以在 Safari 分享菜单中选择“添加到主屏幕”。
